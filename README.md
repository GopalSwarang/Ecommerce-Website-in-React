# 🛒 React E-Commerce Web Application

A modern and responsive E-Commerce Web Application built using React.js. This project demonstrates the implementation of core React concepts such as Components, Props, State Management, React Hooks, Routing, Forms, and Conditional Rendering.

The application allows users to browse products, add items to the cart, manage authentication through Login and Signup pages, and provides an Admin Panel for product management.

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/react-ecommerce-app.git
```

### 2. Navigate to Project Folder

```bash
cd react-ecommerce-app
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Start Development Server

```bash
npm run dev
```

### 5. Open Browser

```text
http://localhost:5173
```
